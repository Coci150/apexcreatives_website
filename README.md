# Apex Creatives Limited - Advertising & Multimedia Website

![Apex Creatives](https://apexcreatives.netlify.app/images/logo.png)

A modern, responsive website for Apex Creatives Limited, a multimedia and advertising agency operating in Zambia, Namibia, and South Africa. The site features a contact form that stores submissions in a Django backend and sends email notifications.
